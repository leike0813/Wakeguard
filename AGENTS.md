## 项目目标

本项目用于生成一个后台服务程序，该服务程序的功能主要有：

- 自动禁用所有支持唤醒电脑的设备的唤醒功能
- 管理一个白名单，在白名单上的设备仍然可以唤醒电脑
- 实时监控是否有新的可以唤醒电脑的设备插入，如果检测到新设备，就提示用户是否需要加入白名单

## 具体要求

1. 该程序应该包装成一个 Windows 服务，在启动时自动启动
2. 设备应有可靠的手段识别其身份，即无论其插入顺序、插入端口，都可以准确唯一确定该设备
3. 需要考虑的设备主要有： 键盘、鼠标等输入设备以及有线网络适配器。如果你知道有其他的设备可以“唤醒”电脑，也可以一并考虑
4. 采用 Rust 语言构建
